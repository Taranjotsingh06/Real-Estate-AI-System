#  Real Estate AI System  

An end-to-end machine learning application that predicts property prices, analyzes market trends, provides insights, and recommends properties based on user preferences.  

This project demonstrates the complete data science workflow — from data collection to model training, analytics, and visualization — showcasing strong hands-on skills in machine learning and data analytics.

---

## Features  

### Price Prediction Module  
- Predicts property prices using XGBoost Regression  
- Considers key features like area, location, bedrooms, bathrooms, and property type  
- Handles complex, non-linear relationships with high accuracy  

---

### Analytics Module  
- Visualizes property price trends across different locations  
- Analyzes how prices vary with different property features  
- Helps users understand market dynamics  

---

### Insights Module  
- Provides feature-wise insights  
- Explains how each feature affects property price  
- Improves interpretability of predictions  

---

### Recommender System  
- Suggests similar or nearby properties  
- Uses similarity-based logic  
- Recommends best-value properties based on user preferences  

---

## API-First Machine Learning Design  

- Built using FastAPI for scalable backend  
- Accepts structured JSON input  
- Performs preprocessing and feature transformation  
- Returns:
  - Predicted property price  
  - Recommendation results  

- Decoupled architecture enables:
  - Web integration  
  - Mobile applications  
  - Future microservices
